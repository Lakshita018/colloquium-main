RULE BOOK

Competition Rules

  • Web-Vida is a dynamic website development competition with a duration of 2 hours.
  • Scheduled for February 17th, 2025, in UG Activity Spaces 1 and 2 at 5:30 PM.
  • The event consists of two rounds.
  • Each team will start with 500 points.

Round 1 :

  • Trivia round on web development-related topics.
  • A QR code will be displayed, redirecting participants to a website containing questions.
  • After completing the round, a leaderboard will showcase the points earned.

Round 2 :

  • Teams will bid for a website domain using the points earned in the trivia round.
  • A QR code will be displayed, redirecting participants to a GitHub repository containing a basic code template.

Website Development Challenge:

  • Teams will be given commands to implement specific features in their website.
  • Completing these tasks will earn extra points.
  • The website-building round will last 45 minutes, after which teams must submit their source files.
  • Judging criteria: creativity, uniqueness, and functionality of the website.

Additional Guidelines:

  • All participants must bring their fully charged laptops with a stable internet connection.
  • The Organizing Committee reserves the right to disqualify teams using unfair means, with the decision being final.
